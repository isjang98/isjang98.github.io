가족을 위한 Android 소리 전환

부모님 전화,
놓치지 않게.

“전화했는데 왜 안 받아?” 하고 보면 진동입니다. 상단바를 쓸어내리는 대신, 부모님은 큰 버튼으로 바꾸고 자녀는 멀리서 상태를 확인합니다.

무료로 다운로드 화면 먼저 보기

엄마: 폰이 안 울린다는데?

나: 진동이에요. 소리로 바꿔드릴게요.

다음부터는 홈 화면 위젯 크게 눌러요.

상황극으로 보면 바로 압니다

문제는 소리 설정이 아니라, 거기까지 가는 길입니다.

작은 아이콘, 빠른 설정, 방해금지 권한. 익숙한 사람에게는 한 번이지만 부모님에게는 매번 설명이 필요한 길일 수 있습니다.

오늘 저녁 7:42

엄마, 전화했는데 왜 안 받으셨어요?

어? 안 울렸는데?

아마 진동으로 바뀐 것 같아요.

그거 어디서 바꾸지?

소리모드 열고 파란 버튼만 누르면 돼요.

설명은 짧게, 버튼은 크게. 상단바를 내리고 작은 아이콘을 찾는 일을 앱 첫 화면과 위젯으로 옮겼습니다.
상단바 수업 끝 “위에서 쓸어내려서…”로 시작하는 설명을 줄입니다.
전화벨 확인 먼저 자녀 폰에서 부모님 폰의 마지막 상태를 보고 요청합니다.
홈 화면에 크게 위젯을 꺼내 두면 앱을 찾는 과정도 한 번 더 줄어듭니다.

소리모드는 “스마트폰을 잘 쓰게 만드는 앱”이 아니라, 꼭 필요한 순간에 덜 헤매게 만드는 앱입니다.

실제 앱 화면

헷갈릴 만한 건 크게, 위험한 건 따로.

소리와 진동은 바로 바꾸고, 무음은 실수하지 않게 별도로 둡니다. 가족 연결도 6자리 코드로만 시작합니다.

소리모드가 하는 일

작은 설정을 가족의 쉬운 루틴으로 바꿉니다.

부모님에게는 “큰 버튼 하나”, 자녀에게는 “상태 확인 후 요청”만 남겼습니다. 그래서 설명이 짧아지고, 실수할 여지가 줄어듭니다.

부모님 폰

지금 소리인지 진동인지 바로 보이게

앱을 열면 현재 상태가 가운데 크게 보입니다. 작은 상태 아이콘을 읽지 않아도 됩니다.

자녀 폰

멀리 있어도 “지금 상태”부터 확인

부모님 폰이 진동인지 확인한 뒤 소리, 진동, 무음 요청을 보낼 수 있습니다.

홈 화면

위젯은 그냥 크게 눌러도 되게

홈 화면에 꺼내 두면 앱을 찾지 않고도 소리와 진동을 크게 눌러 바꿉니다.

안심하고 연결

가족 원격은 허용한 만큼만.

이름, 전화번호, 위치, 사진, 통화 내용은 수집하지 않습니다. 가족 연결과 원격 요청에 필요한 최소 정보만 처리하도록 설계했습니다.

초대 코드 기반 연결 부모님 폰에서 만든 6자리 코드로 직접 승인한 가족만 연결됩니다.
원격 허용 스위치 부모님 폰에서 가족 원격 제어 허용을 끄면 요청은 적용되지 않습니다.
개인정보처리방침 공개 처리 항목과 목적을 문서로 확인할 수 있습니다.
