개인정보처리방침

소리모드 쉬운전환 - 가족 원격 · 시행일: 2026년 06월 26일

소리모드 쉬운전환 - 가족 원격은 사용자가 소리, 진동, 무음 모드를 쉽게 전환하고, 사용자가 허용한 경우 가족이 원격으로 소리 모드 변경을 도울 수 있도록 필요한 최소한의 정보만 처리합니다.

1. 처리하는 개인정보 항목

앱은 서비스 제공을 위해 다음 정보를 처리할 수 있습니다.

  • 기기 식별용 임의 ID: 앱에서 생성하는 무작위 UUID입니다.
  • 역할 정보: 부모님 폰 또는 자녀 폰으로 선택한 앱 사용 역할입니다.
  • FCM 토큰: 가족 원격 요청을 부모님 폰으로 전달하기 위한 푸시 토큰입니다.
  • 현재 소리 모드 정보: 소리, 진동, 무음 상태와 마지막 갱신 시각입니다.
  • 가족 연결 정보: 초대 코드, 가족 연결 상태, 연결 해제 상태입니다.
  • 원격 요청 기록: 요청한 모드, 전송 성공 여부, 실패 사유, 요청 시각입니다.

앱은 이름, 전화번호, 주소록, 통화 내용, 문자 내용, 위치 정보, 사진, 마이크 녹음 정보는 수집하지 않습니다.

2. 개인정보 처리 목적

  • 사용자의 현재 소리 모드 표시 및 소리/진동/무음 전환 기능 제공
  • 홈 화면 위젯을 통한 빠른 소리/진동 전환 기능 제공
  • 가족 연결을 위한 초대 코드 생성 및 확인
  • 사용자가 허용한 경우 가족 원격 제어 요청 전달
  • 부모님 폰의 마지막 상태 확인 및 원격 요청 결과 표시
  • 오류 확인, 부정 사용 방지, 서비스 안정성 유지

3. 개인정보 보관 및 이용 기간

서비스 제공에 필요한 정보는 앱 사용 및 가족 연결이 유지되는 동안 보관됩니다. 가족 연결을 해제하면 연결 상태는 해제 처리되며, 이후 원격 제어 목적으로 사용되지 않습니다. 사용자가 삭제를 요청하는 경우, 법령상 보관이 필요한 경우를 제외하고 확인 후 삭제합니다.

4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 판매하지 않으며, 광고 목적으로 제3자에게 제공하지 않습니다. 다만 서비스 운영을 위해 아래 외부 서비스를 사용할 수 있습니다.

  • Supabase: 기기 등록, 가족 연결, 초대 코드, 원격 요청 처리 및 저장
  • Firebase Cloud Messaging: 원격 요청 푸시 메시지 전달

위 서비스는 각 제공자의 개인정보처리방침과 보안 정책에 따라 데이터를 처리할 수 있습니다.

5. 권한 사용 안내

  • 알림/푸시 관련 기능: 가족 원격 요청을 수신하기 위해 사용됩니다.
  • 방해 금지 모드 접근 권한: 사용자가 무음 전환을 요청할 때만 필요합니다.

소리와 진동 전환은 가능한 한 추가 권한 없이 동작하도록 설계되어 있으며, 무음 전환처럼 권한이 필요한 기능은 필요한 시점에만 안내합니다.

6. 개인정보의 안전성 확보 조치

  • 기기 식별에는 이름이나 전화번호가 아닌 무작위 식별자를 사용합니다.
  • 가족 연결은 초대 코드를 통해 사용자가 직접 승인한 경우에만 생성됩니다.
  • 원격 제어는 부모님 폰에서 가족 원격 제어를 허용한 경우에만 동작합니다.
  • 서버 통신은 HTTPS 기반으로 처리됩니다.
  • 앱 화면에서는 릴리즈 버전에서 FCM 토큰 등 디버그 정보를 표시하지 않습니다.

7. 사용자의 권리

사용자는 언제든지 앱에서 가족 연결을 해제하거나 원격 제어 허용을 끌 수 있습니다.

  • 가족 원격 제어 허용 해제
  • 가족 연결 해제
  • 앱 삭제를 통한 로컬 데이터 삭제
  • 개인정보 삭제 또는 문의 요청

8. 아동의 개인정보

이 앱은 아동을 대상으로 하지 않으며, 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 개인정보처리방침 변경

본 개인정보처리방침은 앱 기능, 관련 법령, 운영 정책 변경에 따라 수정될 수 있습니다. 변경이 있는 경우 앱 또는 배포 페이지를 통해 안내합니다.
