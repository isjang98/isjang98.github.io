⚾ 야구일기 개인정보처리방침

시행일: 2026년 4월 5일

1. 수집하는 개인정보

야구일기는 개인정보를 수집하지 않습니다.

앱에서 입력하는 모든 데이터(직관 기록, 사진, 메모 등)는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 데이터 저장

모든 데이터는 기기 내부 데이터베이스(Room)에 저장됩니다.

  • 직관 기록 (날짜, 팀, 구장, 결과, 스코어, 좌석, 메모)
  • 사진 URI (기기 갤러리 참조)
  • 앱 설정 (응원팀, 온보딩 완료 여부)

앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

3. 분석 도구

야구일기는 앱 사용 패턴을 이해하기 위해 Firebase Analytics를 사용합니다.

수집되는 정보:

  • 화면 조회 이벤트
  • 버튼 클릭 이벤트
  • 앱 사용 빈도

이 데이터는 익명으로 수집되며, 개인을 식별할 수 없습니다. Google의 개인정보처리방침이 적용됩니다.

4. 제3자 제공

야구일기는 사용자의 데이터를 제3자에게 제공하지 않습니다.

5. 아동 개인정보

야구일기는 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
